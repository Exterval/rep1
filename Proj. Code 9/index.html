Introduction to Java

Java is an Object-oriented programming language developed by Oracle.

Advantages of Java include the capability to run on a variety of computers, its security and it is architecturally neutral.

Runs on a JVM(Java Virtual Machine) which allows the conversion of the previously-compiled bytecode to machine-code(binary) which improves performance significantly.

More features of Java:

  • A binary program into which the Java
    • compiler converts source code
  • Java interpreter
    • Checks bytecode and communicates with the operating system
  • Executes bytecode instructions line by line within the Java Virtual Machine
  • WORA = Write once, run anywhere
  • Windowed applications Menus, Toolbars, Dialog boxes

    • Programming

    Programming in java always requires you to type the main function as it is the basis of your program/algorithm.

    Main function is as follows:
    public static void main(String[] args){ }

    public - An access modifier, which allows the JVM to access the main method from anywhere.

    static - will share instances across the class(file) itself.

    void - function not expecting a return value.

    main - starting point of the program.

    String[] args - allows us to type into the program and command line.

    A hello world program in Java would look like this:

    public class HelloWorld {
    public static void main(String[] args) {
    System.out.println("Hello World!");
    }
    }

    The output would be Hello World!

    Said output is located in a console in your chosen IDE(Integrated Development Environment)

    Variables

    The declaration of variables is done for storing data.

    Common data types and declarations are as follows:
    //primitive
    int inte = 100;
    double doub = 1.0;
    float flo = 1.0f; //needs f
    boolean bool = false;
    char chara = 'a';
    short shor = 2;
    byte bit = 1;

    //non primitive
    String string = "John";

    Arithmetic

    Math operations in java contain addition, subtraction, multiplication, division, and modulus.

    Additional math operations need java.math.*; to be declared above their main function.

    An example of a simple arithmetic operation in Java is the following:
    public static void main(String[] args) {
    int sum = 0, x = 1, y = 2;
    sum = x + y; //1+2
    System.out.println(sum);
    }

    Output of the program will be 3.

    Loops

    Loops are useful for repeating a statement in a program.

    Commonly used loops are While, do-While, and for.

    An example of a loop in Java would look like this:

    public static void main(String[] args) {
    // for loop = executes a block of code a limited amount of times.
    for(int i = 10; i>=0; i-=2){ //will decrement 10 until it is no longer greater than or equal to 0
    System.out.println(i);
    }
    System.out.println("Happy new year!");

    The output would look like this:
    10
    8
    6
    4
    2
    0
    Happy new year!

    Arrays

    Arrays are used for storing multiple values in a variable.

    Declaration of arrays may look like this:
    String[] cars = new String[3];

    You now have a variable that can have multiple values.

    Arrays are counted from 0 onwards. This is their index. Same case as characters in a string.

    To assign values in an arrays is the following:

    cars[0] = "Camaro";
    cars[1] = "Courgette";
    cars[2] = "Tesla";

    If you wish to output a specific index of the array, simply declare the array with the given index.

    It would look something like this:
    System.out.println(cars[0]);

    It would produce an output of
    Camaro
